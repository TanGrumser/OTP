One-Time Pad Verschlüsselung

Tipp: Bewege die Maske, um das verschlüsselte Bild zu entschlüsseln

Wie funktioniert's?

Das One-Time Pad ist eine perfekte Verschlüsselungsmethode. Hier siehst du:

  • Ein verschlüsseltes Bild (versteckt)
  • Eine Maske (die du bewegen kannst)
  • Wenn die Maske richtig positioniert ist, wird das Originalbild sichtbar

Bewege die sichtbare Maske. Wo sie das (unsichtbare) verschlüsselte Bild überlappt, siehst du das Ergebnis der Entschlüsselung (XOR-Operation). Außerhalb siehst du nur die Maske.
